html学习01--语法

发表于 2017-06-25 | 分类于 html | | 阅读次数

掌握一门语言最快捷的方法是首先了解他的语法，然后集中时间快速学习这门语言，并写3个以上小项目

标签

  • h标签
    h – 标题标签 一共有6级 h1-h6,h1最大

  • p标签
    p – 段落标签

  • 文字控制标签属性
    color – 文字颜色
    size – 文字字号大小 只有1-7级 7级最大
    face – 文字字体

  • hr 水平分割线标签属性
    color – 颜色
    size – 粗细大小，没有限制
    with – 宽度
    align – 水平对齐方式 有三个值：left左 center居中 right右
    注意：
    标签有那些属性是规定的，没有的不能乱加
    p – 段落
    h – 段落
    p和h标签接受align属性

  • body标签的的属性
    background – 背景图片
    bgcolor – 背景颜色
    背景色永远在背景图片下

  • b u i 标签和空格 换行
    空格的特殊字符： &nbsp
    换行标签：

    b — 加粗效果
    u — 下划线效果
    i — 斜体效果

  • img 图片控制标签 快速插入图片的快捷键
    src — 表示图片的路径（必备）
    with — 图片宽度
    height — 图片高度
    boder — 图片边框
    titile — 鼠标移动到图片上时的提示文本
    alt — 图片不能正常显示时的提示文本
    with — 图片宽度
    with — 图片宽度

列表

  • 无序列表
    基本结构：ul>li;ul是整体结构 li是列表项
    type 样式：
    disc — 小圆点（默认）
    circle — 小圆圈
    square — 小方块

  • 有序列表
    基本结构：ol>li;ol是整体结构 li是列表项
    type 样式：1,a,A,i,I

  • 项目列表（自定义列表）
    基本结构：dl>dt+dd; dl表示列表的整体 dt表示项目（标题）；dd表示详情（内容）；
    项目列表经常用来制作图文混排的结构

选择器

  • css复合选择器
    选择器指的就是选择标签的不同方式
    复合选择器是有基础选择器组成的一类复合选择器

  • 后代选择器
    例：ol li{}表示ol里面所有的li
    .box .li01{}表示.box里面所有类名为li01的标签

  • 并集选择器
    通过逗号隔开表示一起选中，来共同书写样式；
    h2,p,ul li, #span{background: red;}

  • 交集选择器
    选择器之间连续书写，表示选中带有指定类名的那个标签；
    例如：p.box{}表示带有box类名的那个p标签

属性

  • 背景属性
    background 表示背景属性，控制标签的背景颜色，是一个复合属性，有多个值
    基本结构：背景颜色+背景图片+平铺方式+水平位置+垂直位置+图像模式
    举例：background：pink url（images/bg.gif） no-repeat 0 0 fixed;
    平铺方式：no-repeat 不平铺；repeat-x横向平铺； repeat-y纵向平铺；
    位置设置两种写法：1.特殊值：水平left center right 垂直方向值：top center bottom
    2.具体像素值：例：100px 100px
    图像模式：fixed–可以让背景图固定于屏幕，不会随着网页滚动而看不见

  • 行高属性 line-height
    一个重要的应用就是让一行文本在一个盒子垂直居中,设置它的行高等于盒子的高度即可;

  • 浮动属性:float
    它的值有两个:left right(没有center);实现的效果就是让盒子左对齐或者右对齐;

网页布局总结:
网页布局最稳定的就是块级标签的垂直排列;
在一个容器中,同级标签只能采用一种排列方式,要么是上下垂直排列,要么是左右水平排列;

超链接伪类

伪类表示一种状态或功能, 只有在出发的时候才会生效;
:link 表示超链接的默认样式;
:visited 表示访问过后的样式;
:hover 表示鼠标移上去的样式;
:active 表示点击时候的样式;
注意事项:实际项目中,只给a和a:hover写样式
