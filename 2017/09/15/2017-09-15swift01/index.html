swift学习01

发表于 2017-09-15 | 分类于 iOS | | 阅读次数

掌握一门语言最快捷的方法是首先了解他的语法，然后集中时间快速学习这门语言，每天写2个以上小项目

类似OC PCH的创建

1– OC PCH创建

  • newfile->Other->PCH File
  • 设置pch文件的路径: targets->BuildSetting->Prefix Header->设置$(SRCROOT)/文件在工程中的路径

1– swift 宏常量创建

  • New File —> Swift File —>将新建类中的Foundation替换为UIKit

状态栏颜色设置

全工程的颜色都是白色（从app启动就是白色）

  • 第一步：在Info.plist中设置UIViewControllerBasedStatusBarAppearance 为NO
  • 第二步：在info.plist中添加：
    UIStatusBarStyleUIStatusBarStyleLightContent

只是部分控制器需要修改状态栏文字的颜色

  • 第一步：在Info.plist中设置UIViewControllerBasedStatusBarAppearance 为NO
  • 第二步：UIApplication.shared.statusBarStyle = .default

添加pod

  • 打开你的终端 cd到你的工程目录下 输入 pod init 回车之后 你会看到你的工程目录里多了一个podfile文件
  • 双击打开podfile文件 在选择的target里输入你要用的库 例如 pod ‘Alamofire’ 关闭保存
