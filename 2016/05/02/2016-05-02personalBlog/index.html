个人域名博客搭建

发表于 2016-05-02 | 分类于 个人博客搭建 | | 阅读次数

声明：博客用的是利用github pages 搭建服务，博客的所有操作是在Mac下操作的，其它终端请做参考

搭建个人域名博客的七大步：

一、安装博客工具：

  • 安装node.js，用来生成静态页面（node官网：https://nodejs.org）
  • 安装git，用来把博客提交到github pages （Mac下载Xcode后自带git不用再安装）
  • 安装hexo，命令$ sudo npm install -g hexo

推荐（不做操作）：homebrew 是Mac上面用来安装或者卸载软件用的非常方便的一个软件，homebrew官网。

homebrew 的安装需要注意的是 homebrew 安装的时候一定要在 sudo 管理员权限下安装，否则可能遇到写入失败的问题。
安装方法：打开终端输入
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

二、建立站点：

  • 打开终端 cd到你想存放站点的目录下，执行命令：$ hexo init blog
  • cd到创建的blog目录下，执行命令：$ npm install,完成后，blog文件夹的目录如下:
  • 启动本地服务：执行命令：$ hexo s
  • 访问http://localhost:4000，查看是否可以访问成功，若网页可正常显示，则站点创建成功

三、关联Github账号：

  • 去github注册一个账号：https://github.com
  • 注册完账号后点击加号，创建一个新的公共仓库，name的格式一定要写成：你的github名+github.io


  • 终端cd到blog目录下，使用vim命令编辑_config.yml，滑到最底部，将deploy里的内容改为如下：

异常处理：
如果出现：

将
repository: https://github.com/rocksky/rocksky.github.io.git
替换为下面的

  • 生成静态页面： $ hexo g 此时若出现如下报错：

ERROR Local hexo not found in ~/blog
ERROR Try runing: ‘npm install hexo –save’

  • 则执行命令：npm install hexo --save
  • 部署:$ hexo d 若执行hexo d仍报错，则在blog目录下执行：
    $ npm install hexo-deployer-git --save

四、配置SSH

  • 配置github用户名：$ git config –global user.name “rocksky”//GitHub用户名
  • 配置github邮箱：$git config –global user.email “1*9@qq.com”//GitHub注册邮箱
  • 生成ssh密钥：$ ssh-keygen -t rsa -C “xxx@qq.com”(你的github邮箱),配置完成后可以在/Users/sky/.ssh/目录下看到生成id_rsa和id_rsa.pub两个文件
  • 将SSH key添加到Github,打开id_rsa.pub文件，里面的信息即为SSH key，将这些信息复制到Github的Add SSH key页面即可。进入Github –> Settings –> SSH keys –> add SSH key， Title里任意添一个标题，将复制的内容粘贴到Key里，点击下方Add key绿色按钮即可
  • 至此，你可以通过https://你的github用户名.github.io/完美的访问你自己的博客了
  • 接下来，你还有更长的路要走，年轻人！甩甩你那飘逸的长发继续前行吧

五、关联个人域名：

  • 用终端ping出你github的ip（例如：ping www.skyrock90.github.io）

  • 去阿里云什么的购买域名，购买成功后进入控制台，进行域名解析

    选择用到的域名点击解析，首次选择下面网站解析的立即修改

    选择需要解析的域名，点击添加域名解析

  • 记录类型选A 主机记录写@ 运营商写默认 记录值写ping出的ip ，最后点击保存

  • 生成CNAME的文件，不知道怎么生成的可以到这里去下载，将CNAME的内容改为你的域名，保存
  • 添加CNAME解析，记录类型选择CNAME，主机记录写www，记录值填写博客的github访问地址（skyrock90.github.io），保存，至此域名设置完

七、新建博客文章：

  • 终端里cd到blog目录下，执行$ hexo new “文章名字”,这时会在blog/source/_posts目录下生成文章名字.md文件，尽量不要使用中文来命名，方便以后设置超链接跳转
